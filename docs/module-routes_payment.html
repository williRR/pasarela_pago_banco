Methods

(static) POST /charge(req, res) → {ChargeSuccessResponse|ChargeErrorResponse|object|object}

Endpoint para procesar pagos de comercio electrónico. Este endpoint recibe la información de la tarjeta y el monto para procesar el cobro a través del servicio bancario (`processPayment`).
